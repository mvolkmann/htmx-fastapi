Here are the steps I took to create this project.

- `cddev`
- `cd htmx`
- `mkdir htmx-fastapi`
- `cd htmx-fastapi`
- `python -m venv .venv`
- `. .venv/bin/activate`
- `pip install fastapi`
- `pip install jinja2`
- `pip install uvicorn`
- Create all the files in this repo.
- Start the server by entering `./run`
- Browse localhost:5000
